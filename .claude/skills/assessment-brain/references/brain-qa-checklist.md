# Assessment Brain QA Checklist

Use this checklist before finalizing any assessment-brain report. If any item
fails, fix it before delivery.

## Preflight
- Active brand is Justin Ray / 143 Leadership.
- Required source files exist and are current.
- `./assessments/{id}-scored.md` exists and matches the assessment_id.

## Data Integrity
- Eclipse band present (low/medium/high).
- Confidence band present (A-D).
- Top Two rays present (2 items).
- Eclipsed Ray present (1 item).
- Nine-ray scores present (shine/eclipse + usable range).
- Validity flags and acting risk captured.
- Work/Life splits captured (if present in scored data).

## Interpretation Order
- Reports follow: Eclipse -> Confidence -> Overall Pattern -> Light Signature ->
  Eclipsed Ray -> Recommendations.

## Language + Glossary
- Use display terms from 14A_Glossary_DisplayTerms_v2.md.
- First-use definitions included for Ray, Eclipse, Top Two, Light Signature,
  Eclipsed Ray, Confidence Band.
- Avoid banned words: "load" and diagnostic language.
- If Eclipse is high, include state-limited language (non-permanent, context
  specific).
- Tone is direct, warm, and non-shaming.

## Tools + Actions
- At least 3 tools in Tools First.
- Each tool includes: pattern, cue, micro-rep, REPs line, reflection line.
- At least one work rep and one life rep under 5 minutes.
- Reset script present (1 minute).
- 7-day plan and 30-day focus present.
- First Move present in quick read.
- Watch Me trigger map present.
- Bridge reps present (Top Two -> Eclipsed Ray).

## Evidence + Citations
- Any research-backed claim has inline citations.
- Evidence captured via /research-vault and citation IDs recorded in metadata.
- Citations block present (even if empty, say "None used").

## Formatting + Delivery
- Templates used with all required sections.
- Generated date/time present.
- Assessment ID present and consistent across files.
- Metadata JSON validates against assessment-brain.schema.json.
- Metrics use display terms (Energy Ratio, Pressure Signal Count) when shown.
- Work/Life split insight included when available.
